# MineGuard 前端架构设计文档

## 1. 技术选型

### 1.1 核心框架
- **框架**：uni-app x
- **Vue 版本**：Vue 3
- **开发语言**：TypeScript

### 1.2 技术特性
- **响应式设计**：适配多端设备
- **组件化开发**：提高代码复用性
- **路由管理**：统一的页面导航
- **状态管理**：全局状态共享
- **网络请求**：API 接口调用

## 2. 项目结构

```
frontend/
├── pages/           # 页面文件
│   ├── common/      # 通用页面
│   │   ├── login/   # 登录页面
│   │   ├── home/    # 首页
│   │   └── profile/ # 个人中心
│   └── index/       # 入口页面
├── static/          # 静态资源
│   ├── avatar/      # 头像图片
│   ├── images/      # 图片资源
│   ├── logo/        # Logo
│   └── svgs/        # SVG 图标
├── components/      # 组件
├── utils/           # 工具函数
├── api/             # API 接口
├── App.uvue         # 应用入口
├── main.uts         # 主入口
├── pages.json       # 页面配置
├── manifest.json    # 应用配置
└── uni.scss         # 全局样式
```

## 3. 页面设计

### 3.1 登录页面
- **路径**：`pages/common/login/index.uvue`
- **功能**：用户登录、角色选择
- **角色**：管理员、司机

### 3.2 首页
- **路径**：`pages/common/home/index.uvue`
- **功能**：数据概览、快捷操作
- **管理员视图**：
  - 数据概览（车辆总数、在线车辆、今日运量、今日预警）
  - 快捷入口（车辆管理、人员管理、调度管理、实时监控、统计分析、成本管理）
  - 最新预警
- **司机视图**：
  - 任务概览（今日任务、待完成、完成率）
  - 快捷操作（开始任务、车辆状态）
  - 车辆状态

### 3.3 个人中心
- **路径**：`pages/common/profile/index.uvue`
- **功能**：个人信息、设置、关于系统

## 4. 功能模块

## 5. 技术实现

### 5.1 路由配置
- **pages.json**：配置页面路径、导航栏样式
- **tabBar**：底部导航栏
- **uniIdRouter**：登录页面配置

### 5.2 状态管理
- **本地存储**：uni.getStorageSync / uni.setStorageSync
- **组件状态**：Vue 3 ref / reactive

### 5.3 样式设计
- **主题色**：蓝色渐变 (#007AFF → #0056b3)
- **布局**：卡片式布局
- **响应式**：适配不同屏幕尺寸
- **动画**：流畅的过渡效果
