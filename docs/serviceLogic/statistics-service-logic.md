# 统计服务逻辑文档

## 1. 模块概述
统计服务负责系统的统计分析功能，包括行程统计、成本统计、财务报表生成等，为系统提供数据统计和分析能力。

## 2. 核心功能

### 2.1 行程统计
- 行程数据统计：统计行程数量、距离、时长等数据
- 行程状态统计：统计不同状态的行程数量
- 行程趋势分析：分析行程数据的趋势变化
- 行程效率分析：分析行程的效率和优化空间

### 2.2 成本统计
- 成本数据统计：统计各类成本数据
- 成本趋势分析：分析成本数据的趋势变化
- 成本结构分析：分析成本的构成和分布
- 成本优化分析：分析成本优化的空间和方向

### 2.3 财务报表
- 财务数据汇总：汇总财务相关数据
- 财务报表生成：生成各类财务报表
- 财务分析：分析财务数据，提供 insights

### 2.4 总体统计
- 系统数据统计：统计系统整体数据
- 业务指标分析：分析关键业务指标
- 绩效评估：评估系统和业务的绩效

## 3. 主要流程

### 3.1 行程统计流程
1. 收集行程数据
2. 处理行程数据
3. 计算统计指标
4. 生成统计结果
5. 存储统计数据

### 3.2 成本统计流程
1. 收集成本数据
2. 处理成本数据
3. 计算成本指标
4. 生成成本分析
5. 存储成本数据

### 3.3 财务报表生成流程
1. 收集财务数据
2. 处理财务数据
3. 生成财务报表
4. 分析财务数据
5. 存储报表数据

## 4. 服务间调用
- 调用行程服务获取行程数据
- 调用成本服务获取成本数据
- 调用AI服务获取数据分析和优化建议
- 调用Python数据清洗服务（端口8008）进行数据预处理

## 5. 数据存储
- 统计数据存储在MySQL数据库中
- 报表数据存储在MySQL数据库中
- 分析结果存储在MySQL数据库中

## 6. 技术栈
- Spring Boot
- MySQL
- MyBatis-Plus
- RESTful API
