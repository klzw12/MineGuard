# AI服务逻辑文档

## 1. 模块概述

AI服务负责系统的智能分析和决策支持，包括数据分析、报表生成、优化建议等功能，为系统提供人工智能能力。

## 2. 核心功能

### 2.1 数据分析

- 统计数据分析：分析统计服务提供的统计数据
- 成本数据分析：分析成本服务提供的成本数据
- 行程数据分析：分析行程服务提供的行程数据
- 预警数据分析：分析预警服务提供的预警数据
- 驾驶行为分析：分析行程轨迹数据，评估驾驶行为

### 2.2 报表生成

- 财务报表生成：生成财务相关报表
- 统计报表生成：生成统计相关报表
- 成本报表生成：生成成本相关报表
- 自定义报表生成：根据用户需求生成自定义报表

### 2.3 优化建议

- 成本优化建议：提供成本优化的建议
- 行程优化建议：提供行程优化的建议
- 调度优化建议：提供调度优化的建议
- 系统优化建议：提供系统整体优化的建议

### 2.4 智能决策

- 调度决策支持：为调度服务提供智能决策支持
- 预警决策支持：为预警服务提供智能决策支持
- 成本决策支持：为成本管理提供智能决策支持

## 3. 主要流程

### 3.1 数据分析流程

1. 接收数据分析请求
2. 获取相关数据
3. 生成分析提示词
4. 调用AI API进行分析
5. 解析AI响应
6. 返回分析结果

### 3.2 报表生成流程

1. 接收报表生成请求
2. 获取相关数据
3. 生成报表提示词
4. 调用AI API生成报表
5. 解析AI响应
6. 返回报表数据

### 3.3 优化建议生成流程

1. 接收优化建议请求
2. 获取相关数据
3. 生成优化建议提示词
4. 调用AI API生成建议
5. 解析AI响应
6. 返回优化建议

## 4. 服务间调用

- 调用统计服务获取统计数据
- 调用成本服务获取成本数据
- 调用行程服务获取行程数据
- 调用预警服务获取预警数据
- 调用调度服务提供调度建议
- 调用Python数据清洗服务（端口8008）进行数据预处理

## 5. 数据存储

- AI分析结果存储在MySQL数据库中
- 报表数据存储在MySQL数据库中
- 优化建议存储在MySQL数据库中

## 6. 技术栈

- Spring Boot
- MySQL
- MyBatis-Plus
- RESTful API
- AI API (OpenAI, 百度文心一言等)
