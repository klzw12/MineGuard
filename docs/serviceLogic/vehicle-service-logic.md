# 车辆服务逻辑文档

## 1. 模块概述
车辆服务负责车辆的管理，包括车辆信息管理、车辆状态监控、车辆维护管理等功能，为系统提供车辆相关的核心业务支持。

## 2. 核心功能

### 2.1 车辆信息管理
- 车辆注册：添加新车辆信息
- 车辆信息更新：修改车辆信息
- 车辆信息查询：获取车辆详细信息
- 车辆删除：删除车辆信息

### 2.2 车辆状态监控
- 实时状态监控：监控车辆实时位置、速度、油量等状态
- 历史状态查询：查询车辆历史状态记录
- 状态异常检测：检测车辆状态异常并触发预警

### 2.3 车辆维护管理
- 维护计划制定：制定车辆维护计划
- 维护记录管理：记录车辆维护历史
- 维护提醒：根据维护计划发送提醒
- 维护成本统计：统计车辆维护成本

### 2.4 车辆调度管理
- 车辆分配：为任务分配合适的车辆
- 车辆状态更新：更新车辆调度状态
- 调度历史查询：查询车辆调度历史

## 3. 主要流程

### 3.1 车辆注册流程
1. 接收车辆注册请求
2. 校验车辆信息
3. 保存车辆信息到数据库
4. 返回注册结果

### 3.2 车辆状态监控流程
1. 接收车辆状态数据
2. 处理状态数据
3. 检测状态异常
4. 保存状态数据到数据库
5. 触发预警（如有异常）

### 3.3 车辆维护流程
1. 制定维护计划
2. 执行维护任务
3. 记录维护信息
4. 更新车辆状态

## 4. 服务间调用
- 调用预警服务处理车辆状态异常
- 调用调度服务进行车辆分配
- 调用统计服务统计车辆相关数据
- 调用成本服务管理车辆维护成本

## 5. 数据存储
- 车辆基本信息存储在MySQL数据库中
- 车辆实时状态数据存储在Redis中
- 车辆历史状态数据存储在MySQL数据库中
- 车辆维护记录存储在MySQL数据库中

## 6. 技术栈
- Spring Boot
- MySQL
- Redis
- MyBatis-Plus
- WebSocket
