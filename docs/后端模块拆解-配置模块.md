# MineGuard 后端模块拆解 - 配置模块

## 1. mineguard-common-core 模块

### 1.1 模块职责

- 统一异常定义：业务异常、系统异常
- 统一响应封装：Result、PageResult
- 通用工具类：字符串、日期、JSON、加密等工具
- 常量定义：响应码、业务常量、枚举定义
- 基础数据结构：分页参数、排序参数

### 1.2 模块依赖关系

#### 模块间依赖

- **无依赖**：作为基础模块，不依赖其他业务模块
- **被依赖**：被所有其他公共模块和业务服务模块依赖

### 1.3 模块文件结构

```tree

├── pom.xml
└── src/main/java/com/klzw/common/core/
    ├── constant/
    │   ├── ResultCode.java              # 响应码常量
    │   ├── CacheConstants.java           # 缓存常量
    │   ├── RedisKeyConstants.java        # Redis键常量
    │   └── BusinessConstants.java        # 业务常量
    ├── enums/

    │   ├── UserStatusEnum.java           # 用户状态枚举
    │   ├── UserTypeEnum.java             # 用户类型枚举
```

### 1.4 核心功能说明

#### 1.4.1 异常处理机制

- **策略模式**：通过 ExceptionHandlerStrategy 接口定义异常处理策略
- **动态注册**：ExceptionHandlerRegistry 支持动态注册异常处理策略
- **全局处理**：GlobalExceptionHandler 统一捕获和处理所有异常
- **优先级机制**：新注册的异常处理策略优先级更高
- **异常分类**：提供 BaseException、BusinessException、SystemException 等异常类

#### 1.4.2 统一响应封装

- **通用响应**：Result 类支持成功/失败状态和数据返回
- **分页响应**：PageResult 类支持分页数据的统一封装
- **链式调用**：提供便捷的静态方法进行响应构建

#### 1.4.3 通用工具类

- **字符串处理**：StringUtils 提供字符串判断、处理等方法
- **日期处理**：DateUtils 提供日期时间格式化和解析
- **JSON处理**：JsonUtils 基于 Fastjson 提供序列化和反序列化
- **加密工具**：EncryptUtils 提供 MD5、SHA-256 加密
- **数据校验**：ValidateUtils 提供手机号、邮箱、身份证等校验

#### 1.4.4 常量和枚举

- **响应码**：ResultCode 定义系统响应码
- **缓存常量**：CacheConstants 定义缓存相关常量
- **Redis键**：RedisKeyConstants 定义 Redis 键模板
- **业务常量**：BusinessConstants 定义业务相关常量，包括四个用户类型
- **状态枚举**：各种状态的枚举定义，如用户状态、用户类型、车辆状态等

#### 1.4.5 基础数据结构

- **基础实体**：BaseEntity 包含通用字段如 ID、创建时间等
- **分页请求**：PageRequest 封装分页参数，支持默认值和边界检查
- **分页响应**：PageResponse 封装分页结果，包含总条数、页码等信息

## 11. 配置模块版本管理

所有配置模块统一版本管理，在父 pom.xml 中定义版本：

```xml
<properties>
    <mineguard.version>0.0.1-SNAPSHOT</mineguard.version>
    <!-- 依赖版本 -->
    <spring-boot.version>4.0.3</spring-boot.version>
    <lombok.version>1.18.24</lombok.version>
    <fastjson.version>2.0.23</fastjson.version>
</properties>

<dependencyManagement>
    <dependencies>
        <!-- 公共模块依赖 -->
        <dependency>
            <groupId>com.klzw</groupId>
            <artifactId>mineguard-common-core</artifactId>
            <version>${mineguard.version}</version>
        </dependency>
        
        <!-- 第三方依赖 -->
        <dependency>
            <groupId>org.projectlombok</groupId>
            <artifactId>lombok</artifactId>
            <version>${lombok.version}</version>
            <scope>provided</scope>
        </dependency>
        
        <dependency>
            <groupId>com.alibaba</groupId>
            <artifactId>fastjson</artifactId>
            <version>${fastjson.version}</version>
        </dependency>
    </dependencies>
</dependencyManagement>
```
