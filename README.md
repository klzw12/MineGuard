# MineGuard 矿山车辆运输管理系统

## 项目简介

MineGuard是一个专注于矿山车辆运输管理的智能化系统，旨在提高运输效率，确保运输安全，实现精细化管理。

## 项目结构

```tree
MineGuard/
├── docs/                # 项目文档
│   ├── db.sql           # 数据库设计（优化版）
│   ├── need.txt         # 需求文档
│   ├── 架构设计-最终版-backend.md  # 后端架构设计
│   └── 架构设计-最终版-frontend.md # 前端架构设计
├── frontend/            # 前端代码（已实现）
│   ├── pages/           # 页面文件
│   ├── static/          # 静态资源
│   ├── App.uvue         # 应用入口
│   ├── pages.json       # 页面配置
│   └── ...              # 其他前端配置文件
├── backend/             # 后端代码（预留位置）
│   ├── src/             # 后端源代码
│   ├── pom.xml          # Maven配置
│   └── ...              # 其他后端配置文件
├── .gitignore           # Git忽略文件
└── README.md            # 项目说明
```

## 快速开始

### 前端开发

1. 进入frontend目录
2. 使用HBuilderX打开项目
3. 运行到浏览器或真机测试

### 后端开发

1. 进入backend目录
2. 使用IDE打开项目
3. 配置数据库连接
4. 启动后端服务
